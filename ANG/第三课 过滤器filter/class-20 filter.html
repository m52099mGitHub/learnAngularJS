num过滤器

  • {{123}}
  • {{12345.67}}
  • {{12345.678|number}}
  • {{123456789|number}}
  • {{12.3456789|number:2}}

currency 格式化数字为货币形式

  • {{1234|currency}}
  • {{42536.4646|currency}}
  • {{253543|currency:'元'}}
  • {{432536|currency:'RMB'}}

date日期过滤器

  • {{today}}
  • {{today|date:'medium'}}
  • {{today|date:'short'}}
  • {{today|date:'fullDate'}}
  • {{today|date:'longDate'}}
  • {{today|date:'shortDate'}}
  • {{today|date:'shortTime'}}

  • 年

  • {{today|date:'yyyy'}}
  • {{today|date:'yy'}}
  • {{today|date:'y'}}

  • 月

  • {{today|date:'MMMM'}}
  • {{today|date:'MMM'}}
  • {{today|date:'MM'}}
  • {{today|date:'M'}}

  • 日

  • {{today|date:'dd'}}
  • {{today|date:'d'}}

  • 星期

  • {{today|date:'EEEE'}}
  • {{today|date:'EEE'}}

  • 时间 小时

  • 24小时制数字小时{{today|date:'HH'}}
  • 一天中的第几个小时{{today|date:'H'}}
  • 12小时制的数字小时{{today|date:'hh'}}
  • 上午或下午的第几个小时{{today|date:'h'}}

  • 时间 分钟

  • 数字分钟{{today|date:'mm'}}
  • 一个小时内的第几分钟{{today|date:'m'}}

  • 时间 秒

  • 数字秒{{today|date:'ss'}}
  • 一分钟内的第几分秒{{today|date:'s'}}
  • 毫秒数{{today|date:'sss'}}
  • 上午或下午的标志{{today|date:'a'}}
  • 四位时区标志{{today|date:'z'}}
  • {{today|date:'yyyy-MM-dd HH:mm:ss'}}
  • {{today|date:'HH:mm:ss yyyy-MM-dd'}}

lowercase|uppercase

  • {{'HYK'|lowercase}}
  • {{'hyk'|uppercase}}

filter

  • 带K{{users|filter:'k'}}
  • {{users|filter:'1'}}
  • {{users|filter:'1':true}}
  • {{users|filter:'k'}}
  • {{users|filter:'k':false}}
  • 首字母大写:{{users|filter:isCapitalized}}

json

  • {{{'username':'HYK','email':'HYK@163.com'}|json}}

宽度限制:limitTo

  • {{'helloworld'}}
  • 限制字符串输出宽度{{'helloworld'|limitTo:5}}
  • 限制字符串输出宽度{{'this is a test'|limitTo:3}}...
  • 限制数组输出宽度{{['a','b','c','d']|limitTo:2}}...

order by 排序

  • 默认{{userinfo}}
  • 名字排序{{userinfo|orderBy:'name'}}
  • 年龄升序排列{{userinfo|orderBy:'age'}}
  • 年龄倒序{{userinfo|orderBy:'age':true}}
